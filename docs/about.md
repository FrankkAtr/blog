# 关于我

我是北京信息科技大学计算机学院信息安全专业学生付建行，学号为 2025012139。

这个 Blog 用于记录开源软件开发课程实践，以及 GitHub、自动化构建和信息安全相关的学习内容。网站源文件保存在公开仓库中，并通过 GitHub Actions 自动发布到 GitHub Pages。

## 使用的开源工具

- VitePress：将 Markdown 内容构建为静态网站。
- Git：记录文件修改历史。
- GitHub Actions：执行自动构建和部署。
- GitHub Pages：托管并公开访问生成的网站。
